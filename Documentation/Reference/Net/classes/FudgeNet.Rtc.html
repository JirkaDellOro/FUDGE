Class Rtc

Manages a single rtc peer-to-peer connection with multiple channels. FudgeNet.Messages are passed on from the client using this connection for further processing by some observer. Instances of this class are used internally by the FudgeClient and should not be used otherwise.

Jirka Dell'Oro-Friedl, HFU, 2021
