Class ParticleSystem

Holds information on how to mutate the particles of a particle system. A full particle system is composed by attaching a ComponentParticleSystem, ComponentMesh and ComponentMaterial to the same Node. Additionally a ComponentFaceCamera can be attached to make the particles face the camera.
