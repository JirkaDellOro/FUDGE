Class Animation

Animation Class to hold all required Objects that are part of an Animation. Also holds functions to play said Animation. Can be added to a Node and played through ComponentAnimator.

Lukas Scheuerle, HFU, 2019 | Jirka Dell'Oro-Friedl, HFU, 2021
