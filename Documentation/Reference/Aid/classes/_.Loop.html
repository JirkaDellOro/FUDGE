Class Loop

Core loop of a FUDGE application. Initializes automatically and must be started explicitly. It then fires EVENT.LOOP_FRAME to all added listeners at each frame

Static start

  • start(_mode?: LOOP_MODE, _fps?: number, _syncWithAnimationFrame?: boolean): void
    • Defined in Documents/FUDGE/core/build/fudgecore.d.ts:7385

    Starts the loop with the given mode and fps. The default for _mode is FRAME_REQUEST, see LOOP_MODE, hooking the loop to the browser's animation frame. Is only applicable in TIME-modes. _syncWithAnimationFrame is experimental and only applicable in TIME-modes, deferring the loop-cycle until the next possible animation frame.

    Parameters

    • Optional _mode: LOOP_MODE
    • Optional _fps: number
    • Optional _syncWithAnimationFrame: boolean
