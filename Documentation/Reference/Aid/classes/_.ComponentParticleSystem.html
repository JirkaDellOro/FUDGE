Class ComponentParticleSystem

Attaches a ParticleSystem to the node. Works in conjunction with ComponentMesh and ComponentMaterial to create a shader particle system. Additionally a ComponentFaceCamera can be attached to make the particles face the camera.

author

Jonas Plotzky, HFU, 2022
