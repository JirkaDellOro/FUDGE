Class BufferReader

Reader to read data from an array buffer more conveniently. It saves a current offset which is updated when data is read due to its bytelength. despite getSequence it is mostly a copy of the reference: https://github.com/picode7/binary-reader

author

Matthias Roming, HFU, 2023
