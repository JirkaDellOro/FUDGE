Class Time

Instances of this class generate a timestamp that correlates with the time elapsed since the start of the program but allows for resetting and scaling. Supports Timers similar to window.setInterval but with respect to the scaled time. All time values are given in milliseconds

authors

Jirka Dell'Oro-Friedl, HFU, 2019
