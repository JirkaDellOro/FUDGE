Class MeshPolygon

Generate a flat polygon. All trigons share vertex 0, so careful design is required to create concave polygons. Vertex 0 is also associated with the face normal.

            0 
          1╱|╲  4 ...
           ╲|_╲╱ 
           2   3

Jirka Dell'Oro-Friedl, HFU, 2021-2022
